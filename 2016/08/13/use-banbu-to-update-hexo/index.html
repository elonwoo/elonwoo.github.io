use babun to update hexo

Posted on 2016-08-13

安装完 babun 后，尝试用 babun 进行 hexo d 的部署，会报以下错误:

研究了下，是因为 babun 自带 ssh，读取的是 /home/username/.ssh

而我们的 public key 是放在系统的 c/Users/username/.ssh 下的

在 babun 做一个软链接

然后再进行 hexo d， 这下就成功了
