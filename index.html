目标

找到不是 20 或 21 开始，6 位数字开头的行，将这些行与上一行进行合并

处理方案

  1. 将所需数据导出为 txt 文档，或者直接复制成txt文档

  3. 搜索文档里不存在的字符，比如 &lt

  4. 为每个 20 或 21 开始，6 位数字开头的行加上 &lt

  5. 建制宏脚本，进行如下操作

  6. 运行刚刚制作的宏脚本

  7. 去除每行行首的 &lt

  8. 导入 SQL Server 投诉表里的字段名，然后将 txt 文档导入到 MySQL 表里
